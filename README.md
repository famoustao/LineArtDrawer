# SketchMaster - 图片转线稿与自动绘制工具

一款基于 Qt6 + OpenCV 的桌面应用程序，可将任意图片转换为矢量线稿，并自动控制鼠标进行高精度临摹绘制。

## 功能特性

### 核心功能
- **图片导入**：支持 PNG、JPG、JPEG、BMP、TIFF、GIF 格式
- **线稿生成**：基于 Canny 边缘检测和轮廓提取算法
- **矢量导出**：导出为 SVG 格式，方便二次编辑
- **SVG 导入**：支持导入外部 SVG 线稿进行绘制
- **自动绘制**：自动控制鼠标按线稿路径绘制

### 高级绘制功能
- **贝塞尔曲线平滑**：将路径拟合为三次贝塞尔曲线，线条更流畅
- **像素级精确绘制**：逐像素移动鼠标，最高精度临摹
- **5 级速度控制**：从 0.2x 到 5.0x 精细调节绘制速度
- **3 级压感模拟**：轻压（细线）、中压（正常）、重压（粗线）
- **绘制区域选择**：通过 F7/F8 热键自由设定绘制区域

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| F7 | 设置绘制区域左上角（当前鼠标位置） |
| F8 | 设置绘制区域右下角（当前鼠标位置） |
| Ctrl + + | 加速 |
| Ctrl + - | 减速 |
| Ctrl + P | 切换精度模式（普通 / 像素级） |
| Ctrl + C | 循环切换压感级别 |
| 空格 | 停止绘制 |

## 构建说明

### 依赖
- Qt6 (Widgets, Core)
- OpenCV 4.x
- CMake 3.16+

### Linux 构建
```bash
mkdir build && cd build
cmake ..
make -j$(nproc)
```

### Windows 构建
使用 GitHub Actions 自动编译（推荐）：
1. 将代码推送到 GitHub 仓库
2. 确保 `.github/workflows/` 文件夹已上传
3. Actions 会自动编译 Windows 可执行文件

或在本地使用 MSVC + vcpkg：
```batch
build_windows.bat
```

## 使用流程

1. **导入图片**：点击"导入图片"选择要转换的图片
2. **生成线稿**：调整对比度、模糊等参数后点击"生成线稿"
3. **编辑线稿**（可选）：在画布上选择、删除或手绘添加线条
4. **设置绘制区域**：
   - 将鼠标移到目标区域左上角，按 F7
   - 将鼠标移到目标区域右下角，按 F8
5. **开始绘制**：点击"开始绘制"，有 3 秒倒计时准备时间
6. **实时控制**：绘制过程中可用快捷键调节速度和精度

## 项目结构

```
SketchMaster/
├── src/
│   ├── core/          # 核心算法（图像处理、线稿生成、SVG 导入导出）
│   ├── ui/            # 用户界面（主窗口、画布、控制面板）
│   ├── mouse/         # 鼠标控制（移动、绘制、区域选择）
│   └── input/         # 热键管理
├── resources/         # 图标等资源
├── CMakeLists.txt     # CMake 构建配置
└── .github/workflows/ # GitHub Actions CI/CD
```

## 技术细节

- **边缘检测**：OpenCV Canny 算法 + 轮廓提取
- **路径简化**：Douglas-Peucker 算法 + 贝塞尔曲线拟合
- **鼠标控制**：Windows SendInput API / Linux X11
- **高精度绘制**：微秒级定时 + 密集采样（最高 4 样本/像素）
